# Elipses 📐

Este repositorio contiene el motor de cálculo y renderizado para figuras elípticas y cónicas. Es una pieza esencial para la visualización avanzada de datos donde la simetría circular se transforma para representar proyecciones en perspectiva o trayectorias dinámicas.

## 🚀 Propósito del Proyecto

Desarrollar algoritmos precisos para la generación de elipses, permitiendo el estudio de proyecciones cónicas y la visualización de datos en espacios deformados. Este módulo es clave para proyectos que requieren representar órbitas, perspectivas de grafos y modelos matemáticos de alta complejidad.

## 📁 Estructura del Repositorio

Siguiendo el estándar profesional de organización:

* **src/**: 
  * Módulos de cálculo de focos, excentricidad y ejes (mayor/menor).
  * Implementaciones en Visual Basic 6 para el dibujo de curvas de Bézier y elipses por puntos.
* **docs/**: Teoría sobre la geometría de las cónicas y su aplicación en la visualización de proyecciones estereográficas.
* **results/**: Modelos generados, pruebas de trazado y exportaciones de coordenadas elípticas.

## 🛠️ Aplicaciones Técnicas

* **Astronomía y Matemáticas**: Base para modelar conocimientos astronómicos (como los de la civilización Maya) que utilizan trayectorias elípticas.
* **Proyecciones Avanzadas**: Complemento al estudio de la **Hipótesis de Riemann**, permitiendo visualizar la "deformación" de la distribución de primos al cambiar el ángulo de proyección.
* **Gráficos en VB6**: Optimización de la función `Circle` de VB6 para generar elipses con control total sobre la rotación y el aspecto.
